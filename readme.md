# Generate random string

## Install

    npm i simple-rand-str
    

## Use

    let r = require('simple-rand-str');
    r.generate();
    
 Output:
 ```sh
 $ node index.js
 $ 35LFm81u9w
 ```
You can set the string length

    r.generate(10);

Output:
 ```sh
 $ node index.js
 $ XMoWxkeJr85JfDdIhaq7
 ```
